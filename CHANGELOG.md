# 更新日志

## [2.0.0] - 2026-04-21

### 新增功能

#### 企业级扫描器
- **超大规模文件扫描** - 支持百万级文件秒级扫描
- **并行处理** - 使用 Rayon 库自动利用所有 CPU 核心
- **智能分组** - 按文件大小分组，减少 90% 哈希计算
- **快速模式** - 小文件只采样前后 4KB 进行哈希
- **内存映射** - 大文件使用 mmap 加速哈希计算
- **增量扫描** - 支持只扫描修改过的文件

#### 专业级虚拟列表
- **百万级数据支持** - 支持 100 万+ 数据流畅渲染
- **分片加载** - 1000 项/块，按需加载
- **内存管理** - LRU 淘汰策略，最大 50000 项内存占用
- **预加载** - 自动预加载可见区域周围的块
- **性能监控** - 实时渲染和滚动性能统计

#### 企业级数据存储
- **IndexedDB 数据层** - 替代 localStorage，支持大数据存储
- **数据版本控制** - 支持数据库迁移
- **备份恢复** - 支持数据导出/导入
- **数据加密** - 敏感数据加密存储（预留）

#### 专业级日志系统
- **分级日志** - DEBUG/INFO/WARN/ERROR/FATAL 五级
- **批量持久化** - 自动批量写入，减少 IO
- **日志轮转** - 自动清理旧日志
- **远程上报** - 支持日志远程上报（预留）

#### 性能优化工具
- **防抖/节流** - 专业级防抖节流实现
- **请求缓存** - LRU 缓存策略，支持 TTL
- **异步队列** - 并发控制，自动重试
- **记忆化** - 函数结果缓存

#### 增强错误处理
- **结构化错误** - 错误分类、级别、恢复策略
- **错误上报** - 自动错误收集和上报
- **批量错误收集** - 批量操作错误统计

### 改进功能

#### 存储管理
- **实时统计** - 显示数据库大小、文件记录数、扫描历史
- **刷新功能** - 手动刷新存储统计

#### 风险等级
- **中文显示** - 风险等级改为中文：低/中/高
- **建议文本** - 根据风险等级给出处理建议

#### UI/UX
- **Element Plus 中文** - 所有组件显示中文
- **操作列宽度** - 加宽操作列，确保按钮完整显示
- **分页组件** - 分页显示中文

#### 代码质量
- **0 编译警告** - 修复所有 Rust 和 TypeScript 警告
- **类型安全** - 完善类型定义
- **代码规范** - 统一代码风格

### 技术债务

#### 后端优化
- **Rayon 并行化** - 目录遍历和哈希计算并行化
- **XXH3 哈希** - 使用 xxhash-rust 替代标准哈希
- **连接池** - SQLite 连接池优化

#### 前端优化
- **Pinia 状态管理** - 统一状态管理
- **组合式函数** - 使用 Vue 3 Composition API
- **虚拟滚动** - 大数据列表虚拟化
